About Cancer Diagnosis Prediction System

Overview

This is an AI-powered cancer diagnosis prediction system that uses machine learning to predict cancer types based on clinical and genomic data. The system is built with Flask and scikit-learn, providing a user-friendly interface for medical professionals and researchers.

Available Models

Implemented Models
  • Logistic Regression - Fast, interpretable baseline
  • Random Forest - Ensemble method with feature importance
  • Gradient Boosting - Sequential ensemble learning
  • XGBoost - Optimized gradient boosting with GPU support
  • Ridge Classifier - Linear model with regularization
  • Support Vector Machine - Kernel-based classification
  • K-Nearest Neighbors - Instance-based learning
  • Naive Bayes - Probabilistic classifier
Model Capabilities
  • Single & batch predictions
  • Probability estimates
  • Feature importance analysis
  • Model interpretability
  • CSV file upload support
  • Decision scores

Features

Fast Predictions

Get instant predictions for single samples or batch process CSV files

Model Reliability

Confidence scores and probability estimates for each prediction

Interpretability

Understand model decisions through feature importance visualization

How to Use

  1. Select a Model: Choose from the available trained models
  2. Provide Input: Either enter JSON data for single predictions or upload a CSV file
  3. Get Results: View predictions with confidence scores and probabilities
  4. Analyze: Check feature importance for model interpretation

Data Requirements

For single predictions, provide feature values as a JSON object:

{
  "feature_1": 1.5,
  "feature_2": 2.3,
  "feature_3": 0.8
}

For batch predictions, upload a CSV file with features as columns and samples as rows.

Dataset

This system is trained on the MSK Redefining Cancer Treatment dataset from Kaggle, which contains genomic and clinical data for cancer classification. The dataset includes:

  • Genetic variants and mutations
  • Clinical text descriptions
  • Multiple cancer types
  • Train/validation/test splits

API Endpoints

Endpoint Method Description
/api/models GET List available models
/api/predict POST Make single prediction
/api/upload-predict POST Batch predictions from CSV
/api/feature-importance/<model> GET Get feature importance
/api/model-info/<model> GET Get model information
/api/model-stats GET Get all models statistics

Technical Stack

  • Backend: Flask, scikit-learn, pandas, numpy
  • Frontend: Bootstrap 5, Chart.js
  • ML Models: sklearn, XGBoost, Joblib
  • Deployment: Python 3.9+
